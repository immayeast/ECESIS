# Solution Workspace

This folder contains the working deliverables for Assignments 1, 2, and 3.

Each assignment follows the same structure:

- `notebooks/`: exploratory analysis and reproducible notebook workflows.
- `execution_outputs/`: generated notebook outputs, including graphs, tables, and intermediate artifacts.
- `solutions/`: final code, scripts, or solution files.
- `summary_reports/`: written summaries, findings, and submission-ready reports.
